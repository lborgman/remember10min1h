Flashcards 4 Internet (fc4i)

Open

Fc4i is yet another app for saving links to web pages you are interested in (by using share, , from the web page). There are some very good apps for that (like Google Keep). So why another one?

Tools for learning: My reason for writing Fc4i was to add integrated tools for learning, like flashcards, repeating reminders, etc. I also included some tools to stimulate thinking: mindmaps, network view.

Reminders in this app are not alarms. They are more like "can you please tell me now if there is something I ought to repeat now?" There is a good description here: The Importance of Reminders

Some things to notice
Storing Apps like Google Keep has advanced storing where your data is backed up to your Google account. Fc4i just stores your data in your web browser.
Native app vs web app The fc4i app and its companions are a progressive web apps (PWA). Google Keep is a native app.

While I install a native app from Google without hesitating I rather avoid installing a native app from some unknown developer (like me). The reason is security concerns. A PWA is essentially just a web page. So it is normally not more dangerous to use than a web page.

Another advantage of web apps is that they should work everywhere where there is an uptodate web browser. (But sadly, sharing to a web app is yet not standardised enough. It probably only works on Android mobiles and tablets.)

Mindmaps 4 Internet (mm4i)

Open

When I first heard about mindmaps I thought it looked like a very good idea. But I never tried to use it. Until one day I talked with a woman studying the same subject as I at the university. She said that she could read a book in one hour and remember it with the help of mindmaps.

I still did not try mindmaps since I never had enough big papers and colored pencils with me. But I was determined to try.

Then one day I saw that there was a Javascript library for creating mindmaps, jsMind. Fantastic! It can be used in any modern web browser. I decided to make something that you could use on a mobile phone built on that Javascript library. That became mm4i.

Mm4i is integrated with fc4i. You can easily link to subjects in fc4i in a mindmap you create with mm4i. However, mm4i can be used alone. No need to use fc4i. (You can link to web pages in nodes in mindmaps if you want to.)
